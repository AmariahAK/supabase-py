Supabase Python Client Guide
=============================

A task-oriented guide for the Supabase Python client (`supabase-py`_).
Learn how to create clients, manage authentication, query your database,
work with storage and edge functions, subscribe to realtime changes,
and test your application.

.. _supabase-py: https://github.com/supabase/supabase-py

Installation
============

Requirements:

- Python >= 3.9

**With pip:**

.. code-block:: bash

   pip install supabase

**With uv:**

.. code-block:: bash

   uv add supabase
